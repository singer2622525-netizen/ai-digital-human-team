## 📝 手动方式（如果脚本有问题）

### 使用Python脚本

在项目根目录创建并运行：

```python
import os
import subprocess

# 切换到web_interface目录
os.chdir('03-技术实现/02-功能模块/web_interface')

# 安装依赖
subprocess.run(['pip3', 'install', '-r', 'requirements.txt'])
subprocess.run(['pip3', 'install', 'cryptography'])

# 启动服务
subprocess.run(['python3', 'app.py'])
```

保存为 `start.py`，然后运行：
```bash
python3 start.py
```
