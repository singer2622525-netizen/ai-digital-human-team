# 环境变量管理指南

## 文件结构
```
~/DeveloperConfig/
├── 环境变量/
│   ├── .env.global
│   ├── .env.work
│   └── .env.personal
└── 脚本工具/
    └── env_manager.py
```

## RAGFlow配置（新增）

在 `~/.env.work` 文件中添加以下配置：

```bash
# RAGFlow知识库配置
# 公司Dell服务器上的RAGFlow服务地址
RAGFLOW_BASE_URL=http://your-server-ip:port

# RAGFlow API密钥（如果需要认证）
RAGFLOW_API_KEY=your-api-key-if-needed

# 软件工程事业部数字人队伍知识库ID
RAGFLOW_KB_ID=your-knowledge-base-id
```

### 配置说明

1. **RAGFLOW_BASE_URL**
   - RAGFlow服务的完整URL
   - 示例: `http://192.168.1.100:9380` 或 `https://ragflow.company.com`
   - 在公司内网访问时使用内网IP
   - 在家里访问时可能需要VPN或公网地址

2. **RAGFLOW_API_KEY** (可选)
   - 如果RAGFlow启用了API认证，需要提供密钥
   - 在RAGFlow Web界面中生成API密钥

3. **RAGFLOW_KB_ID**
   - 在RAGFlow中创建的"软件工程事业部-数字人队伍"知识库的ID
   - 在RAGFlow Web界面的知识库详情页可以找到

### 加载环境变量

在终端中运行：
```bash
source ~/DeveloperConfig/环境变量/.env.work
```

或者在Python代码中：
```python
from dotenv import load_dotenv
load_dotenv('~/DeveloperConfig/环境变量/.env.work')
```
