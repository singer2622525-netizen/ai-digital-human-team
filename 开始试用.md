## 🚀 快速开始（3步）

### 第一步：安装依赖

```bash
# 进入Web界面目录
cd "03-技术实现/02-功能模块/web_interface"

# 安装依赖
pip3 install -r requirements.txt

# 安装密码数据库依赖
pip3 install cryptography
```

### 第二步：启动Web界面

```bash
# 在web_interface目录下
./start.sh

# 或者
python3 app.py
```

**访问地址**：`http://localhost:5001`

### 第三步：体验功能

1. **浏览Web界面**
   - 首页：查看数字人状态
   - 任务管理：创建和管理任务
   - 可视化：查看任务依赖关系

2. **使用密码数据库**（可选）
   ```bash
   cd "03-技术实现/05-配置管理"
   python credentials_manager.py add-service RAGFlow --type 知识库 --url https://ragflow.suuntoyun.com
   ```

3. **查看项目进度**
   ```bash
   cd "03-技术实现/02-功能模块"
   python3 progress_tracker.py
   ```

## ⚠️ 注意事项

### 1. 端口占用

如果5001端口被占用：
```bash
# 检查端口
lsof -ti:5001

# 关闭占用进程
lsof -ti:5001 | xargs kill -9
```

### 2. Ollama（可选）

如果使用AI功能，需要：
```bash
# 安装Ollama
brew install ollama

# 启动服务
ollama serve

# 下载模型
ollama pull deepseek-coder:6.7b
```

### 3. 主密码

密码数据库首次使用需要设置主密码，请记住这个密码。
