
### 方法2：手动执行（使用绝对路径）

```bash
# 安装依赖
cd "/Users/a1/Documents/Projects/cursor/00-00软件工程事业部（AI 数字人队伍）/03-技术实现/02-功能模块/web_interface"
```

### 方法3：使用Python脚本

在项目根目录创建 `start.py`：

```python
import os
import subprocess

# 切换到web_interface目录
web_dir = "03-技术实现/02-功能模块/web_interface"
os.chdir(web_dir)

# 安装依赖
print("📦 安装依赖...")
subprocess.run(['pip3', 'install', '-r', 'requirements.txt'])
subprocess.run(['pip3', 'install', 'cryptography'])

# 启动服务
print("\n🚀 启动Web服务...")
print("📍 访问地址: http://localhost:5001")
subprocess.run(['python3', 'app.py'])
```

然后运行：
```bash
python3 start.py
```
