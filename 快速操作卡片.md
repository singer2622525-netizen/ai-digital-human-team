# 🚀 快速操作卡片

## 🏠 在家里Mac（工作结束）

### 关闭系统
```bash
./关闭系统.sh
# 或
lsof -ti:5001 | xargs kill -9
```

### 同步数据（可选）
```bash
# 在Web界面中：
# 1. 打开 http://localhost:5001
# 2. 点击"🔄 数据同步"标签
# 3. 点击"推送到GitHub"同步代码
# 4. 选择存储类型，点击"同步"同步数据库
```

---

## 🏢 在公司Mac（开始工作）

### 启动系统
```bash
cd "03-技术实现/02-功能模块/web_interface"
./start.sh
```

### 访问系统
```
http://localhost:5001
```

### 同步数据（在Web界面中）
1. 打开 http://localhost:5001
2. 点击"🔄 数据同步"标签
3. 点击"从GitHub拉取"同步代码
4. 选择存储类型，点击"同步"同步数据库

---

## 📋 常用命令

| 操作 | 命令 |
|------|------|
| 启动系统 | `cd "03-技术实现/02-功能模块/web_interface" && ./start.sh` |
| 关闭系统 | `./关闭系统.sh` |
| Git拉取 | `git pull origin main` |
| Git推送 | `git push origin main` |
