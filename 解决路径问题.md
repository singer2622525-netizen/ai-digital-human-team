# 🔧 解决路径问题

## ⚠️ 问题说明

从终端提示符 `a1@1s-Mac-mini ~ %` 可以看出，你当前在**home目录**（`~`），而不是项目目录。

所以运行 `./安装依赖.sh` 时找不到文件。

---

## ✅ 解决方案

### 方法1：切换到项目目录（推荐）

在终端中运行：

```bash
cd "/Users/a1/Documents/Projects/cursor/00-00软件工程事业部（AI 数字人队伍）"
```

然后运行：

```bash
./安装依赖.sh
./启动Web界面.sh
```

```bash

```

### 方法3：使用bash执行


## 📋 完整操作步骤

