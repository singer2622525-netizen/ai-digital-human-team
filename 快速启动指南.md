# 🚀 快速启动指南（解决中文路径问题）

## ⚠️ 问题说明

如果遇到"目录不存在"或"文件找不到"的错误，可能是中文路径在终端中的编码问题。

## ✅ 解决方案

### 方案1：使用项目根目录的启动脚本（推荐）

在项目根目录下运行：

```bash
# 1. 安装依赖
./安装依赖.sh

# 2. 启动Web界面
./启动Web界面.sh
```

### 方案2：使用绝对路径

```bash
# 进入项目根目录（使用绝对路径）
cd "/Users/a1/Documents/Projects/cursor/00-00软件工程事业部（AI 数字人队伍）"

# 进入web_interface目录
cd "03-技术实现/02-功能模块/web_interface"

# 安装依赖
pip3 install -r requirements.txt
pip3 install cryptography

# 启动服务
python3 app.py
```

### 方案3：使用Python脚本

```bash
# 在项目根目录
python3 << 'EOF'
import os
import subprocess

# 切换到web_interface目录
os.chdir('03-技术实现/02-功能模块/web_interface')

# 安装依赖
subprocess.run(['pip3', 'install', '-r', 'requirements.txt'])
subprocess.run(['pip3', 'install', 'cryptography'])

# 启动服务
subprocess.run(['python3', 'app.py'])
EOF
```
